# Plugin Development Guidelines

> Conventions for creating plugins distributed through this marketplace.

---

## Overview

Plugins are the units of distribution in this marketplace. Each plugin can contain **skills** (slash commands), **agents**, **hooks**, **MCP servers**, and **LSP servers**. These guidelines cover how to author each component type.
